Synapse AI - Your Smart Meeting Assistant ✨

Transform hours of long meetings into crisp, actionable summaries in seconds! No cap, it's that easy. 

Synapse AI is a production-ready **Full-Stack SaaS application** designed to record, upload, transcribe, and summarize meetings. Built with a focus on an aesthetic, glassmorphism UI and a highly scalable backend architecture.

### 🛠️ Tech Stack & Magic Behind It:
* **Frontend:** React.js, Tailwind CSS (Fairy Aesthetic & Glassmorphism vibes 🌸)
* **Backend:** Node.js, Express.js, MongoDB (The classic MERN stack magic)
* **Architecture:** AWS S3 Presigned URLs for seamless large file uploads
* **Processing:** Redis & Bull Queues for asynchronous AI background tasks
